TSP Optimization Using Genetic Algorithms

Implemented heuristic solutions for the Traveling Salesperson Problem...

This project showcases advanced implementations of the Traveling Salesperson Problem (TSP) using Genetic Algorithms and graph theory techniques. The TSP is a well-known optimization challenge where the goal is to determine the shortest possible route that visits a set of cities once and returns to the origin city.

Key Highlights:
  • Randomized population generation.
  • Tournament selection for parent selection.
  • Order-1 crossover and mutation for genetic diversity.
  • Elitism to retain optimal routes during optimization.
Tools and Technologies:
  • Python
  • NumPy, Matplotlib
  • Genetic Algorithms

This project demonstrates the power of heuristic approaches for solving complex optimization problems with real-world applications in logistics and route planning.
